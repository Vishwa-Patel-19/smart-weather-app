# 🌦 Smart Weather App
A Python CLI weather app that fetches real-time weather data using API.

## 🚀 Features
- Search weather by city name
- Shows temperature
- Shows weather condition
- Shows humidity
- Saves weather reports in file

## 🛠 Technologies Used
- Python
- Requests Library
- API Integration

## ▶ How to Run
1. Install requests:
pip install requests
2. Run:
python main.py

## 📌 Example Output
City: Goa
Temperature: 33 C
Condition: Sunny
Humidity: 54
